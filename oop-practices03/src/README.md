### 객체지향프로그램 연습 : 학점계산기 프로그램 
- 객체들간의 관계를 고민
- 동적인 객체를 정적인 객체로 추상화
   : 이수한 과목 - 객체지향프로그램. 자료구조 등등을 이수했다고 가정 => "과목" 이라는 클래스로 추상화
- 이수한과목을 전달하여 평균학점 계산 -- 요청 --> 학점 계산기 Class --> (학점수 * 교과목평점) 의 합계
                                                                --> 수강이수학점 총 함계
  => 학점수와 교과목평점을 알고 있는 Course(과목) Class 에게 요청
